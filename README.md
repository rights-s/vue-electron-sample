# vue-electron-sample

## Project setup
```
npm install yarn
npx yarn
```

### Compiles and hot-reloads for development
```
npx yarn electron:serve
```

### Compiles and minifies for production
```
npx yarn electron:build
```

### Run your unit tests
```
npx yarn test:unit
```

### Lints and fixes files
```
npx yarn lint
```
